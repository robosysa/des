How to Activate your License Key

You can manage your license keys by logging in to your customer profile on our website.

IMPORTANT: Your log-in password and username details were sent to the enetered email address, after your first purchase. You can find your license key in the Purchase Confirmation email, which was sent after completed purchase.

» Click here to Log-In Now «
Instructions:
  1. Log in to your customer profile
  2. Enter your domain name and license key in appropriate input fields.
  3. Make sure to only use lower-case characters!
  4. Click on Activate button to add the domain name, or Deactivate button to remove it.
  5. Your entered License Key and Domain should now display inside License Overview panel.
  6. After your license key is connected to your domain, you will need to enter it inside the "Home" panel of the admin dashboard.

Note: It is important to connect your domain with a license key as the landing page will function properly only on connected domains.

Uploading files

After downloading the .zip archive file which contains your purchased landing page files, you will need to upload them to your hosting server. To do so, you will need to transfer all the files from downloaded .zip archive file, inside the root directory of your hosting server which is located inside www or public_html folder (both are the same folder).

You can read more about uploading the files to your server on this url.

Updating files to a newer version

In order to update the landing page to a newer release version, you need to extract all the files from downloaded .zip archive of a newer relase version and replace all of the existing files inside of your installation folder where landing page is located.

Before replacing the files you should create a backup of your current db.sqlite3 database file which is located inside the /includes folder. After you update the landing page files, copy your backup file db.sqlite3 back inside the /include folder to avoid any data loss.

IMPORTANT: Create a backup of your current db.sqlite3 file which is located inside /includes folder to avoid any loss of your existing data.

You can find more information about updating your landing page on this url.

Accessing the admin panel

After you have uploaded the landing page files to your hosting server, you can access the admin panel. Admin panel dashboard can be accessed by navigating to the admin subfolder appended to your domain name.

EXAMPLE: If your domain name is called mydomain.com, then the admin dashboard would be accessed by entering the following URL: mydomain.com/admin/

You can find more information about accessing the admin dashboard on this url.

First Login

For your first dasboard access, you can login using the default admin credentials which are showed on the log-in form. After you log-in using the default admin details, you will be re-directed to change the default admin username and password. Please make sure to use a safe password which only you have access to.

Upon changing the admin username and password, you will be re-directed back to the login form where you can login using the new admin details which you just entered.
